### 编码说明

---

> 项目编号 -> id
>
> 项目名称 -> item
>
> 所属关系 -> include
>
> * 全体：0
> * 学生：1
> * 老师：2
>
> 图标编码 -> font
>
> 图标颜色 -> color
>
> 是否显示 -> display
>
> * 显示：true
> * 隐藏：false
